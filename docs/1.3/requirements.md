# Requirements

Server requirements:

* PHP 5.3+
* Apache with mod_rewrite or Nginx web server
* SQLite, MySQL, or PostgreSQL database
* PHP CURL extension required for some features

Browser requirements:

* Google Chrome
* Safari 4+
* Firefox 3.5+
* IE 8+
